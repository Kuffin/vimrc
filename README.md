Vimrc
=====

Features
--------

* very good documentation
* lots of useful key short-cuts, Macros, Commands etc.
* easy customizable
* custom status-line
* allows user-specific ~/.vimrc
* allows project specific vimrc ;)


Used Plugins
------------

* Spell
* Clang_Complete
